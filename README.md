# 📊 Retail Sales Data Analysis Project

## 🧾 Problem Statement

A retail company sells electronic products such as laptops, mobiles, headphones, and keyboards through stores located in different cities like Pune, Mumbai, and Nagpur.

Every purchase made by a customer is recorded in a sales dataset containing:

* Order details
* Product information
* Quantity sold
* Price

Recently, management observed fluctuations in sales but lacks clarity on:

* Which products are driving revenue
* Which cities are performing better

### 🎯 Business Questions

This project aims to answer:

1. Which products generate the most revenue?
2. Which cities have the highest sales?
3. Which product categories perform best?
4. What are the sales trends over time?

---

## 🚀 Solution / Features

This project uses **SQL, Excel, and Python** to analyze retail sales data and generate meaningful insights.

### 🔑 Key Features

* Store and manage sales records (order details, product, city, quantity, price) using Excel/SQL
* Analyze **product-wise revenue** to identify top-performing products
* Evaluate **city-wise sales performance** to determine high-revenue locations
* Compare **category-wise performance** (Electronics vs Accessories)
* Analyze **sales trends over time** using order dates
* Perform data aggregation:

  * Total sales
  * Revenue
  * Quantity sold
* Create visualizations:

  * Bar charts
  * Pie charts
  * Line graphs
* Use Python for advanced data analysis
